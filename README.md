# 🌍 GeoTrack - Location Visualization App

GeoTrack is a full-stack application that enables users to visualize and interact with real-time location data on an interactive map.

## 🚀 Live Demo
👉 https://geotrack-khaki.vercel.app

---

## 🛠 Tech Stack

**Frontend**
- React.js
- Leaflet (React-Leaflet)
- JavaScript, CSS

**Backend**
- .NET 8 Web API (RESTful APIs)

**Deployment**
- Vercel (Frontend)
  
**Backend API**
- Base URL: https://geotrack-backend-aqg6.onrender.com  
- Projects API: /api/projects  
- Swagger: /swagger

## 🔌 API Integration

- Fetches location data from .NET REST APIs
- Handles asynchronous requests using fetch
- Uses a configurable API base URL for backend communication (defined in code)

## ✨ Features

- 📍 Interactive map rendering using Leaflet
- 🔗 Integration with REST APIs for dynamic data
- ⚡ Responsive and performant UI
- 🌐 Deployed frontend with environment-based configuration
- 🔄 Real-time data visualization
